Локальный Rule 34. Local_R34

Удобный поиск и просмотр изображений и видео скачанных из Rule34 с помощью Ruxx (https://github.com/trickerer01/Ruxx)

Возможности:
- Поиск по тегам
- Прога сама делает для себя превьюшки
- Нарезает видосы на кадры, и делает для них живые превью (прям как на порносайтах)
- 7 красивых тем
- Моментальное копирование нужного арта куда надо.
- А самое главное: РАБОТАЕТ КУДА БЫСТРЕЕ ЧЕМ ПРОВОДНИК ВИНДЫ.

Использование:
1. Выбери корневую папку, в которой лежат папки авторов и сами работы
2. Просканируй папку.
3. Нажми две кнопки сверху, чтобы прога сама для себя сделала превьюхи заранее. (можно и не делать это, но так прога будет каждый раз нарезать превью только при поиске)

Ну в правом блоке сам видишь че за кнопки, не маленький, разберёшься.
